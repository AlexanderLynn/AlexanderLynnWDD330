Arrays, Logic and Loops

  • the "delete" operator can be used to remove items from arrays. Example: delete avengers[3];
  • Arrays can be "destructured" toa llow for fast swapping. This was introduced in ES6 to avoid making temporary variables. Example: [x,y] = [y,x];
  • Javascript has many different methods for quickly manipulating arrays. Contained within them are methods such as pop, push, splice, and join
  • Like other C-syntax languages, Javascript also supports multi-dimensional arrays
  • Maps were also introduced in ES6. They function similarily to an array of key and value pairs. The first being the lookup key, the second being the mapped value. For example:
    const romanNumerals = new Map(); romanNumerals.set(1,'I').set(2,'II').set(3,'III').set(4,'IV').set(5,'V');
    romanNumerals.get(4); --> 'IV'
  • Weak Maps were also introduced in ES6. They are similar to Maps, but their keys cannot be primitives and the garbage collector will remove dead entries.
  • Another shrothand way of writing an if else statement is witht the ternary operator. Example: const n = 5; n%2 === 0 ? console.log('n is an even number') : console.log('n is an odd number');
    --> 'n is an odd number'
  • When working with arrays, ES6 introduced the for-of loop. It is similar to the foreach loop in other C-syntax langauges. Example:
    for(const value of avengers){
    console.log(value);
    }

Javascript function

  • In Javascript, functions are written very similarily as to other C-syntax languages. Functions are declared with the function keyword and must include parentheses.
    This is true during declaration as well as calling.
  • As introduced in ES6 (much of what is new was introduced then with small modificatiosn in ES7) the "rest" operator cna be used when an unknown amount of arguments is
    pass into a function. For example:
    function rest(...args){
    for(arg of args){
    console.log(arg);
    }
    }
    rest(2,4,6,8);
    --> 2
    4
    6
    8
  • Javascript also included ways of establishing default arguments if nothing was passed into the function. For example:
    function hello(name='World') {
    console.log(`Hello ${name}!);
    }
    Would produce the result of "Hello World!"
  • ES6 also introduced arrow functions. These are much les verbose than previous function declarations. Example: const hello = () => alert('Hello World!');
  • Javascript includes hoisting within it's interpreter. This means that the function ro variable can be called before it is declared. Functions are always hoisted.
  • When a variable declared with var is called before it is defined, the call will return undefined. When called before is declaration with let or const, it results in an error.
  • Callbacks are also powerful alternative ways to call a function within another function. Example: if(typeof(callback) === 'function'){
    callback();
    }
  • Super useful array method arrayName.filter(); Returns only the items that return true to the comparator inside of the parantheses. For example:
    numbers.filter(x => x%2 === 0 ); // this returns true if the number is even. Only the even numbers are returned from the statement.
